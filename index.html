Congratulations! You are an ideal candidate for learning Python! Python is a fun little language, lacking a lot of the characteristics of most well-known programming languages, such as ending lines with semicolons and declaring variables with their type right at the get-go. Indentation is important, but you won't see very much punctuation from a beginning program. Additionally, it reads very well to the human eye, making reading over someone else's code a breeze.

Congratulations! You are an ideal candidate for learning C++! C++ is an extension of the C programming language. It was designed for system programming, and can be found in video game programming, web servers, and sometimes space probes. C++ can be quite powerful, but can also feel confusing to new users. The first time you see "std::cout << a;" might throw you for a loop, but it very quickly becomes familiar.

Here are some resources to learning C++:

  • https://www.learncpp.com/
  • https://www.codecademy.com/learn/learn-c-plus-plus
  • https://www.programiz.com/cpp-programming

Congratulations! You are an ideal candidate for learning JavaScript! The functionality of this very website was made largely with JavaScript (and of course, HTML and CSS). JavaScript, not to be confused with Java, is one of the core backbones of our current internet. Almost everything that a website can functionally do, regarding things higher level than just displaying text, is probably designed with JavaScript. You should visit your favorite site without JavaScript on and see what breaks! Remember to fix it when you're done...
